Skills and Jobs

In order to implement Easy Perf Check as a minimum viable product (MVP) and launch it to the marker, the following positions need to be filled.

Network Engineer

Position Description

  • Design, build and implement the network projects and solutions based on business and client requirements
  • Respond to work requests, maintain changes to the network so that it operates more efficiently. This includes repairing, improving, upgrading or increasing functionality of the network
  • Be innovation and implement upgrades to the network
  • Work collaboratively within a team

Requirements

  • Tertiary qualifications in Information Technology, Software Engineering or Computer Science
  • Minimum 5 years’ experience in a network engineering role

Skills

  • Strong problem solving and troubleshooting skills to ensure technical problems are managed efficiently and effectively
  • Excellent verbal and written communication skills
  • Effective team management skills and attention to detail

Technical Skills

  • Routing/switching
  • Firewalls/network security for Cisco ASA’s and VPN technologies
  • Wireless
  • Cisco collaboration
  • Network virtualization

User Experience/Interface Designer

Position Description

  • Work independently to design, develop and maintain new and existing websites.
  • Continuously improve visual and user experience design of existing websites.
  • Ensure design is user-friendly and work with content creators with the planning and implementation process.

Requirements

  • Tertiary qualifications in Information Technology, Software Engineering or Computer Science
  • Minimum 3 years’ experience in a website design role

Skills

  • Ability to work independently without supervision
  • Strong communication skills to work within a team
  • Experience with Agile approach
  • Attention to detail

Technical Skills

  • Lean UX, Lean Startup and HCD principles
  • Basic and deliver wireframes
  • Advanced understanding of HTML5, CSS3 and JavaScript and experience of working with developers to create internet sites
  • Adobe creative suite
  • JavaScript Libraries/frameworks
  • Understanding of usability and UX principles
  • Sketch, Figma, Adobe XD

Full Stack Developer

Position Description

  • Write code for digital solutions to enhance our client experience.
  • Your work will include new and existing web apps, new and existing native mobile apps that add value to our students, teachers and stakeholders.
  • Develop new tools and services for our clients to use
  • Updating and maintaining website platforms to maintain contents
  • Integrate with other service providers using their APIs
  • Update and maintain documentation for all development systems and workflows

Skills

  • Agile thinking
  • Ability to prioritise tasks
  • Strong communications skills to work collaboratively within a team
  • Attention to detail
  • Problem solving skills

Technical Skills

  • Experience with writing code using PHP Laravel, Vue.js/Javascript, REACT, Git, SQL, CSS, SSH & HTML5
  • Microsoft Visual Studio Code
  • Knowledge of API best practices
  • Understanding of UI principles, mobile UI conventions and responsive design techniques
  • Experience with working on back-end projects

Business Analyst

Position Description

  • Analyse data to identify business growth opportunities
  • Collect and analyse data on client surveys to identify weakness and design strategies for improvement
  • Make timely decisions and high quality data insights by establishing comprehensive analytical frameworks
  • Push operations and product strategies to be implemented by collaborating with necessary resources
  • Continuous improvement on usage of local tools
  • Manage the monitoring, maintenance, development, upgrade and support of IT system for servers, PCs, operating systems, software applications.

Requirements

  • Bachelor’s degree in IT or Business

Skills

  • Experience in supporting the decision making in teams
  • Knowledge of programming languages such as Hive, Python, Shell and analytical tools such as SPSS and visualization tools
  • Advanced database/SQL and Excel skills
  • Ability to analyse complex problems and propose comprehensive solutions in clear, concise manner
  • Working in agile environment
  • Teamwork to work collaboratively to work with cross function team members
